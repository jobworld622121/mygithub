## 📝 고객 요구사항 정리
1. 상품 구매 과정 중(아이 쇼핑-장바구니-구매) 웹페이지 모바일 환경 최적화
2. 상품 사진 부각
3. 추가한 장바구니 속 상품의 개수 실시간 반영
4. 테스트 URL 공유


## 📋 기능 목록
1. 상품 리스트
2. 상품을 2개의 컬럼으로 보여준다.
3. 장바구니에 ‘담기’ 버튼과 ‘담김’ 버튼 구분
4. 상품 목록 페이지 상단의 장바구니에 담긴 상품 개수 실시간 반영
5. 상품 총 개수 표시 (현재 -개의 상품이 있습니다.)
6. 무한 스크롤
